# Smart QQ Java

基于 Smart QQ（Web QQ） 的 Api 封装，你可以用这个 Api 制作属于自己的 QQ 机器人！

注：由于 Smart QQ 不支持收发图片等功能，所以此 Api 也只可以发送文字消息（不包含@命令）

### Api 列表

如果你想要了解 Web QQ 的通讯方式，并自己实现一个通讯接口。我在 Wiki 中详细的记录了抓包获取的请求和对应参数信息。欢迎参考。
